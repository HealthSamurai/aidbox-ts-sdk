# Aidbox TS SDK

Projects for building interfaces with TypeScript.

## React components

[Storybook](https://healthsamurai.github.io/aidbox-ts-sdk/react-components/)

- `pnpm storybook`

### Hot reload

- `git clone git@github.com:HealthSamurai/aidbox-ts-sdk.git`
- `cd aidbox-ts-sdk && git checkout design-update && pnpm link`
- `cd <your-ui-project> && pnpm link @health-samurai/react-components`
- `pnpm install`
